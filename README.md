# VariablesYOperadores
Respuesta Teoricas.
¿Para qué se utilizan los operadores logicos en programación?
Se utilzan para evaluar condiciones y tomar decisiones dentro de un programa.
¿Por qué es importante declarar correctamente el tipo de dato de una variable?
Porque nos permite trabajar correctamente con la información, evitar errores y hacer el código más claro y ordenado.
